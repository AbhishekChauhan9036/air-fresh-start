Course Overview

This course provides complete knowledge of Face Recognition Technology using Artificial Intelligence, Machine Learning and Deep Learning techniques. Students will learn how to detect faces, extract facial features, train models and deploy real-time face recognition systems.

Course Objectives

  • Understand fundamentals of Artificial Intelligence and Machine Learning
  • Learn Image Processing basics
  • Understand Face Detection vs Face Recognition
  • Work with OpenCV and Python
  • Train Deep Learning models
  • Build real-time Face Recognition system
  • Deploy Face Recognition projects
